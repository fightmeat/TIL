# 문제
아래 예제와 같이 새싹을 출력하시오.

# 입력
입력은 없다.

# 출력
새싹을 출력한다.

# 예제 입력 1 
# 예제 출력 1 
             ,r'"7
    r`-_   ,'  ,/
     \. ". L_r'
       `~\/
          |
          |
# Solution
- 문자열 안에 특수 문자(예: 백슬래시 \, 따옴표 ', ")를 사용하려면 이스케이프 문자를 사용
- 따옴표를 문자열 안에 포함시키려면 \" 또는 \'를 사용하고, 백슬래시를 문자열 안에 포함시키려면 \\를 사용한다.
```python
print('         ,r\'"7')
print('r`-_   ,\'  ,/')
print(' \\. ". L_r\'')
print('   `~\\/')
print('      |')
print('      |')
```
